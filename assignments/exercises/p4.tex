\subsection*{Exercise 1}
Overview the documentation of the Python package \textit{SpacePi}, located in \url{http://spacepy.lanl.gov/}.

\subsection*{Exercise 2}
Install NASA's JPL Horizons On-Line Ephemeris System Python API, which is available on \url{https://github.com/mihok/horizon-jpl} and run the demo available in its source code. It is better to void using pip, try instead to manually install it following the instructions placed in the previous URL.

You can find an application example on \url{http://nasa.apphb.com/explore}.

\subsection*{Exercise 3}
Open The Python Standard Library reference manual\footnote{\url{https://docs.python.org/3/library/index.html}}, and browse the packages installed by default.

\begin{itemize}
\item Which package would you use to download a file with HTTP?
\item Locate at least one package to develop graphical applications with Python.
\end{itemize}

\subsection*{Exercise 4}
Python includes a command-line tool named \texttt{pip3} (just \texttt{pip} in Python 2.X) that makes installation of new packages a piece of cake. All the packages available for this tool are listed in \url{https://pypi.python.org/pypi?\%3Aaction=browse}. Locate, and, if you have root permissions, install and validate the installation the packages to perform the following tasks:

\begin{itemize}
\item Plot scientifical data.
\item Develop a videogame.
\item Twitter data analysis.
\item Predict a planet's position.
\end{itemize}

\subsection*{Exercise 5}
If you have root permissions, install the package named \texttt{astroML}, which applies Machine Learning to Astronomy. Be careful with dependences.

\begin{enumerate}
\item Select, download and run any of the examples listed in \url{http://www.astroml.org/examples/datasets/index.html}
\end{enumerate}

\subsection*{Exercise 6}
We are going to create a new package with two modules and then use them.

\begin{enumerate}
\item In your working directory, create a new folder named \texttt{test}
\item Create a package initialization file \texttt{test/\_\_init\_\_.py} which just prints out that the packages is being initialized
\item Repeat the same with modules \texttt{test/module1} and \texttt{test/module2}, each with its own initialization script.
\item Define a simple function in \texttt{test/module1/foo.py} and \texttt{test/module2/foobar.py}
\item Import package \texttt{test} and execute the functions placed in \texttt{foo.py} and \texttt{foobar.py}
\end{enumerate}
